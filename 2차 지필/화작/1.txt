언어 공동체에는 특정한 방식의 의사소통이 반복되면서 형성된 고유한 담화 관습과 작문 관습이 있음
ㄴ> 주제나 내용, 주제나 내용을 다루는 방식, 사용하는 어휘, 내용의 조직 방식이나 표현 방식, 관습적으로 인정되는 전제와 금기

기능 2. 화자나 청자, 필자나 독자의 태도를 해석하는 일정 기준으로 작용

화법과 작문을 효과적으로 수행하기 위한 방법
언어 공동체에 따라 구별되는 담화 관습과 작문 관습을 올바르게 이해, 적용
ex) 1. 법조인들로 구성된 언어 공동체는 법률과 관련된 내용을 다루고 법률 용어를 사용하는 담화 관습, 판결문이라는 형식적 작문 관습에 익숙
    2. 젊은 세대는 감정을 직설적으로 표현, 줄임말을 많이 사용

유의점 1. 한 사람이 다양한 언어 공동체에 속할 수 있다는 것을 염두
유의점 2. 자기가 속한 언어 공동체의 담화 관습과 작문 관습을 이해하고 적용
유의점 3. 청자나 독자가 속한 언어 공동체의 담화관습과 작문 관습도 고려



바람직한 언어문화의 발전에 기여하는 태도
사회·문화적 상화에 따라 지속적으로 변화
이유 1. 언어 공동체의 담화 관습과 작문 관습을 비판적으로 수용해야함 -> 신중하게 검토
이유 2. 자기가 속한 언어 공동체의 담화 관습과 작문 관습이 현재의 사회·문화적 상화에 부합하지 않아 의사소통에 방해가 된다면 수동적인 태도로 수정, 보완
이유 3. 담화 관습과 작문 관습이 바람직하다면, 그것을 개방적인 태도로 수용

변화해도 언어 생활에서 잃어버리면 안되는 것 : 진실성, 공손성
진실성 : 말을 하거나 글을 쓸 때 질실한 내용을 전달하기 위해 노력
공손성 : 상대방을 존중하는 마음으로 정중한 표현을 사용
이유 4. 태도 - 진실성과 존중성을 바탕으로 의사소통해 나간다면 바람직한 언어문화 형성해 나가는 데 기여 가능

가치가 있는 정보의 선별 방법
1. 정보가 글의 목적에 맞는지, 글의 주제를 전달하는 데 적절한지를 따져 봐야함
ㄴ> 독자의 요구나 기대, 관심, 배경지식에 부합하는지 확인
2. 자료에 포함된 정보가 지나치게 오래되어 효용이 떨어지는 판단 -> 최신의 정보를 수집, 선별
3. 정보를 수록하고 있는 자료가 신뢰성있는지 판단
ㄴ> 전문가인지, 공신력있는 기관인지 확인, 정보가 과정되거나 왜곡되지는 않았는지 검토


선별한 정보의 조직 방법
정보를 전달하는 글의 기본 구조
처음 : 정보를 전달하려는 이유와 목적 밝힘, 전달할 내용 개관
중간 : 중심 내용과 세부 내용 제시하는
끝 : 앞에서 설명한 내용을 요약 or 정리


1. 시간 순서에 따른 조직 방법  
정보가 시간의 흐름에 따라 변화 or 일련의 과정으로 구성될 때 사용
ex) 인물의 생애, 건국 과정
시간의 순서를 차례대로 따르는 순행적 조직 방법 / 시간의 순서를 거슬러 올라가는 역행적 조직 방법

2. 공간 순서에 따른 조직 방법
공간의 이동에 따라 변화하거나 구별되는 특징이 있을 때 사용
ex) 지리산 둘레 길에 대한 정보 제시 -> 길을 걸을 때 거치는 공간의 순서대로 정보 조직
가까운 곳 -> 먼곳 , 위 -> 아래 , 중앙 -> 주변부

3. 논리적 순서에 따른 조직 방법
수집한 정보가 사물이나 현상의 본질과 특성을 설명하는 것일 때 사용
ex) 사막화 현상과 관련하여 화석 연료의 온실가스 배출량 증가, 물 부족 현상 등을 수집했다면 논리적 순서인 원인과 결과의 관계에 따라 조직 가능

4. 비교, 대조 - 유사성 비교, 차이점 대조

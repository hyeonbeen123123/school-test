지식의 종류
선언적 지식(명제적 지식) : 사실, 개념, 원리 등에 대한 지식  ex) 서울은 대한민국 수도이다. 
절차적 지식 : "절차 또는 방법을 통해서 이루어지는가?"에 관한 지식, ex)달걀 프라이를 만드는 법
조건적 지식 : 특정 조건으로 문제를 해결하기 위해 선언적 지식 + 절차적 지식 합친 것? ex) 수학문제

지식을 표현할 때
내용 단순화, 명확, 완전하게

서술(글) 외에 지식을 전달하는 방법
수식 -> 그림 ex) 조선시대 때 현대 2차방정식 풀이를 위해 그림 이용
지식 -> 수학 식 ex)만유인력 공식

시멘틱 네트워크
노트의 집합 + 링크의 집합 // 링크 : 노드 사이의 관계를 표현

시멘틱 네트워크의 단점을 보완 -> 프레임




추론 : 명제 및 술어 논리를 이용하여 지식을 표현한 다음, 이를 활용하여 새로운 지식을 만드는 것
연역적 추론 : 이미 알고 있는 사실 -> 새로운 사실로 유도하는 방법
귀납적 추론 : 관찰된 사실 -> 일반적인 원리를 유도하는 방법



데이터 : 특정 분야에서 수집됐지만, 그 자체로는 의미x, 가공x
정보 : 데이터를 가공한 것 
지식 : 정보를 모아서 분석하여 얻은 결과
지식의 종류 ; 선언적 지식, 절차적 지식, 조건적 지식

표현된 지식 활용
명제 논리와 술어 논리로 표현된 지식을 활용하여 새로운 지식 도출


문제 정의 -> 데이터 수집 -> 데이터 전처리 -> 탐색적 데이터 분석 -> 모델 학습 -> 모델 평가 -> 공유와 배포
데이터 전처리 + 탐색적 데이터 분석 = 데이터 탐색
데이터 탐색 : 학습 돌리기 전까지 하는 짓 ex) 라벨링, 전처리 등등
